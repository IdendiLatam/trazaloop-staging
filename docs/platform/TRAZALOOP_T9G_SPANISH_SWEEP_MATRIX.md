## 4. Control permanente

`npm run test:t9g-spanish` deja el detector §19 (texto JSX, labels,
placeholders, aria-label, title, alt, mensajes error/message de Server
Actions, con la allowlist técnica) y la verificación de cero «organización»
visible como parte de `test:all`. Es un control complementario: no reemplaza
la revisión manual por rutas y componentes registrada arriba.
